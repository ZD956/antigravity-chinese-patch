# Antigravity 2.0 完美中文汉化补丁 | 支持一键极速安装与无痕还原

🎉 **专为最新版 Google Antigravity 2.0 打造的非破坏性全界面中文本地化补丁。**

本补丁通过动态注入技术对 IDE 进行汉化，**不修改主程序的任何核心逻辑代码**，安全稳定。它拥有高度可扩展的本地词库，能够覆盖从基础菜单、弹出对话框到深层开发者设置（如沙盒外命令、MCP 模型上下文协议）的各个角落！

---

## ✨ 核心特性 (Features)

- ⚡ **一键极速安装**：包含一键安装批处理脚本（`.bat`），全程自动化检测与注入，无脑傻瓜式操作，只需几秒钟即可畅享中文界面！
- ♻️ **无痕安全还原**：包含一键无痕还原脚本。基于修改 preload 层实现，不对底层的渲染逻辑进行硬编码替换。即便 IDE 更新或你想卸载补丁，也能随时一键无缝还原为官方原版英文。
- 👁️ **Shadow DOM 穿透**：完美支持 Web Components 深度嵌套节点中的文本捕获与翻译，无死角覆盖所有新式前端组件界面。
- 🧩 **支持正则表达式提取**：内置强大的正则捕获和替换功能（例如：动态邮箱提示、特定参数的后缀捕捉等）。
- 📝 **高度可定制的本地词库**：翻译数据全部保存在 `locales/zh-CN/webui.json` 中，任何用户都可以用普通的文本编辑器轻松补充和修改自己的词条。

## 🚀 安装指南 (Installation)

### 环境准备
- 你的电脑上需要安装有 **Python 3.8+**，且已添加到系统的 `PATH` 环境变量中。

### 安装步骤
1. **彻底关闭 Antigravity 2.0**（确保后台没有其相关进程）。
2. 将本项目克隆或下载到本地，解压后打开 `antigravity-chinese-patch` 文件夹。
3. 双击运行 **`一键安装.bat`**。
4. 等待黑色控制台窗口提示“汉化补丁安装完毕！”，按任意键退出。
5. 重新打开 Antigravity，即可看到全中文的界面啦！

## 🛠️ 自定义与扩展词库 (Customization)

如果您在使用中发现了少量未被汉化的英文，您可以非常轻松地自行补充：

1. 打开 `locales/zh-CN/webui.json` 文件。
2. 找到 `"exact"` 字典：
   ```json
   "exact": {
     "Open File": "打开文件",
     "Some New Text": "一些新文本"
   }
   ```
3. 按照格式增加您的英中对照词条（注意英文要和界面上**完全一致**，包括空格和大小写）。
4. 保存文件，重启 Antigravity，新词库会自动生效，**无需再次执行安装脚本！**

---

## ♻️ 卸载与无痕还原 (Uninstallation)

如果遇到兼容性问题，或者想还原纯正的英文原版：

1. 彻底关闭 Antigravity。
2. 双击运行文件夹内的 **`一键还原.bat`**。
3. 提示成功后重启 IDE，即可恢复最初的原版英文状态。

## ❓ 常见问题 (FAQ)

**Q：安装后打不开 IDE（白屏或崩溃）怎么办？**  
A：一般是因为主程序的 `app.asar` 在解包/打包时遇到了权限问题或损坏。请运行 `一键还原.bat` 进行恢复。如果是 IDE 版本过新导致的 DOM 结构大改，请前往 Issues 提交反馈。

**Q：有些地方还是英文，为什么？**  
A：由于 IDE 界面的文案可能会随着版本动态下发，词库可能没有完全涵盖最新版本的所有内容。您可以通过修改 `webui.json` 的方式自己补充，也欢迎向本仓库提交 Pull Request 来分享您的汉化成果！
